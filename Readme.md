Application to know the value of a cryptocurrency with the currency of your country.

V1
What the tool does:
The tool does the conversion of your country's currency with the selected cryptocurrency.

v1.1 Under development
Online calculator of the currency of your country with the selected cryptocurrency, placing the full value to be exchanged, special for exchange houses.

v1.2 Planned
Development of the tool for mobile devices 

Stack used for development
The application was created with React using Vite, the api of https://cryptocompare.com was used, using the Hooks useState and useEffect, API consuo with async - awayt
